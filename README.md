# 📊 Unemployment Analysis using Python

## 📌 Project Overview
This project analyzes unemployment data across India to identify trends, patterns, and the impact of COVID-19.

## 🧠 Key Features
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Visualization of unemployment trends
- COVID-19 impact analysis

## 🛠️ Tools & Technologies
- Python
- Pandas
- Matplotlib
- Seaborn

## 📈 Visualizations
- Unemployment Trend Over Time
- Heatmap of Unemployment Rates
- Top States with Highest Unemployment

## 📂 Files Included
- `unemployment_analysis.py` → Python code
- `Unemployment in India.csv` → Dataset
- `trend.png`, `heatmap.png`, `top_states.png` → Visual outputs

## 🚀 Conclusion
This project helps understand how unemployment changed over time and highlights the impact of COVID-19.
